# RF Integrated Circuits (EE6320) Spring 2026

This repository contains the project submissions for the course EE6320 : RF Integrated Circuits instructed by Prof. Sankaran Aniruddhan in the Jan-May 2026 semester at IIT Madras. Brief description of each project is provided below -

**Project 1** - Design of a Low Noise Amplifier for 3.45 GHz - 3.55 GHz.

**Project 2** - Design of a Gilbert Cell Mixer for RF Frequency of 3.45 GHz to 3.55 GHz.

**Project 3** - Design of a Voltage Controlled Oscillator for 6.90 GHz to 7.10 GHz operation, integrated with frequency divider with output 3.45 GHz to 3.55 GHz operation, and with variable phase output for L and Q. 

**Project 4** - Design of a Power Amplifier for 3.45 GHz - 3.55 GHz.
